⏱️ Time Tracker
A lightweight, visually elegant personal time tracking and efficiency analysis tool. It helps you record where every minute goes and empower self-improvement through data.

How to Use:
Extract the compressed package, then double-click Start Software.bat to launch the program.


📝 Time Tracking
Quickly record daily start/end times and work content.
12 preset activity types for one-click selection: Value Work, Communication, Dining, Review & Reflection, Entertainment, Professional Learning, AI Application, Cutting-edge Information, Capital Operation, Errands Handling, Routine Tasks, Strength Training.
Supports adding, editing and deleting records with simple and intuitive operations.
📊 Data Statistics
Check time allocation from daily, monthly and yearly dimensions.
Visual pie charts display the time proportion of each activity type.
Combined statistics by activity type and work content, accurate to every specific task.
Time range filtering to flexibly view data of any time period.
🤖 AI Intelligent Analysis
Connects to DeepSeek API to generate personalized analysis reports based on your real time records.
Deeply analyze time usage patterns and identify efficiency bottlenecks.
Provide 5-7 targeted and actionable time management suggestions.
Supports customized system prompts and user prompt templates.
📊 Excel Export
One-click export of all records into formatted .xlsx files.
Automatically set header styles, column widths and alignment; files are directly usable after export.
🎨 Modern Interface
Desktop: Tkinter-based modern design with rounded buttons, gradient colors and hover animations.
Web: Cyberpunk-style HTML dashboard, supporting online viewing and Excel export.
Remembers window size and position for automatic restoration on next startup.
