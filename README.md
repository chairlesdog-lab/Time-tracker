# TimeTracker Pro - User Guide
## 🚀 Getting Started
1. Double-click 启动软件.bat to launch the application
2. Wait for the web server to start and the browser to open automatically
## 📝 Recording Your Time (Tab 1)
1. Set Time : Enter start and end dates/times (supports 24-hour format)
2. Choose Activity : Select an activity type from the buttons or dropdown
3. Add Details : Fill in what you did in the content box
4. Save : Click the save button to record your activity
## 📊 Viewing Statistics (Tab 2)
1. View your time distribution as a colorful pie chart
2. See detailed breakdowns by activity type
3. Export your records to Excel if needed
## 🤖 AI Analysis (Tab 3)
1. First, set your DeepSeek API key in the settings
2. Select a date range to analyze
3. Click "Analyze" to get AI-powered insights and suggestions
4. Save or review your analysis results
## 💾 Data Management
- Your records are automatically saved locally in time_records.json
- To clear all data, delete time_records.json and config.json (be careful!)
- Your window position and settings are saved automatically
## 💡 Tips
- Update your end time to the current time for accurate tracking
- Use specific content descriptions for better AI analysis
- Regularly check the statistics tab to spot time-wasting patterns
That's it! Start tracking and take control of your time! ⏰✨
